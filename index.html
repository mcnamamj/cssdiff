cssdiff – Calculate semantic CSS difference via url parameters

When you allow user-modified CSS and want to create a diff between the original and their modified version via URL parameters/request parameters, you can use this tool to handle it all in-browser!

Calculate!

This is a tool, that can help you to calculate a difference between 2 CSS payloads. Urlencode both the old and new CSS data and pass them as the URL parameters old and new respectively.
